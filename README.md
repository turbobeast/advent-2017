## Troubleshooting IDE

- Make sure you have [reason-cli](https://github.com/reasonml/reason-cli) installed for current version of node. 

- Install merlin and ocp-indent
```bash
opam install merlin
opam instll ocp-indent
```

- Configure your env

```bash
opam switch
eval $(opam config env)
```
